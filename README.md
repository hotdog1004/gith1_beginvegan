## 국비교육 1차 프로젝트 : beginvegan
#### 채식주의자와 비채식주의자들을 위한 채식주의 레시피 사이트입니다.
-------------------
#### 1. 프로젝트 기획 이유 

환경에 대한 의식이 변화하면서 채식주의를 지향하는 사람들이 늘고있습니다. 
반면에 채식주의 레시피만을 공유하는 채식주의 중심적인 사이트는 존재하지 않아 이 프로젝트를 기획하게 되었습니다.

- 조리단계별 댓글과 레시피 댓글 CRUD
